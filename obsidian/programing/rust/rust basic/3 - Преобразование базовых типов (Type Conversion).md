#rust #теория
Rust требует полной ясности при работе с числами. Нельзя использовать тип `u8`, работая с типом `u32`: это ошибка.

К счастью, в Rust есть ключевое слово **as**, позволяющее очень легко преобразовывать типы.
```rust
fn main() {
    let a = 13u8;
    let b = 7u32;
    let c = a as u32 + b;
    println!("{}", c);

    let t = true;
    println!("{}", t as u8);
}

```
