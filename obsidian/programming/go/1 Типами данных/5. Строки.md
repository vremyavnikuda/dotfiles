# Строки в Go

Go поддерживает строки как последовательности символов Юникода:

## Объявление строки
```go
var s string = "Hello, World!"
```

## Конкатенация строк
```go
s1 := "Hello, "
s2 := "World!"
s := s1 + s2
```

## Доступ к символам строки
```go
char := s[0]  // 'H'
```

## Пример использования строк
```go
package main

import "fmt"

func main() {
    var s string = "Hello, World!"
    fmt.Println(s)

    s1 := "Hello, "
    s2 := "World!"
    s = s1 + s2
    fmt.Println(s)

    char := s[0]
    fmt.Println(char)
}
```
