# Неявная типизация в Go

Go поддерживает неявную типизацию с использованием оператора `:=`:

## Объявление переменной с неявной типизацией
```go
a := 42
b := 3.14
c := "Hello, World!"
d := true
```

## Пример использования неявной типизации
```go
package main

import "fmt"

func main() {
    a := 42
    b := 3.14
    c := "Hello, World!"
    d := true

    fmt.Println(a)
    fmt.Println(b)
    fmt.Println(c)
    fmt.Println(d)
}
```
