#java #javacore #переменные #примерыкода 

```md
- byte - целые числа (самый маленький, обычно 8 бит)
- short - целые числа (меньше, обычно 16 бит)
- int - целые числа (средний, обычно 32 бит)
- long - целые числа (большой, обычно 64 бит)
- float - действительные числа (меньше, обычно 32 бит)
- double - действительные числа (больше, обычно 64 бит)
- char - символьный тип (хранит один символ)
- boolean - логический тип (хранит значение true или false)
- String - строковый тип (хранит строки текста)
```
