#rust 
Для отладки кода на Rust вы, возможно, часто используете println!(). Но есть альтернативный макрос - dbg!(), который может сделать вашу отладку ещё проще и информативнее!

```rust
fn main(){
	let var1=2;

	println!("{}",var1);
	dbg!(var1);
	dbg!(var1*2);
}

//Результат
[srs/main.rs:5] var1=2
[srs/main.rs:5] var1*2=4
```
